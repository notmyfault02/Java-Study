# 스레드 상태

## 스레드의 상태
* 객체 생성
> 열거 상수: NEW    
스레드 객체가 생성, 아직 start() 메소드가 호출되지 않은 상태

* 실행 대기
> 열거 상수: RUNNABLE    
실행 상태로 언제든지 갈 수 있는 상태

* 일시 정지
> 열거 상수: WAITING    
다른 스레드가 통지할 때까지 기다리는 상태

> 열거 상수: TIMED_WAITING   
주어진 시간 동안 기다리는 상태

> 열거 상수: BLOCKED    
사용하고자 하는 객체의 락이 풀릴 때까지 기다리는 상태

* 종료
> 열거 상수: TERMINATED    
실행을 마친 상태
